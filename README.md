# Sprint 11 — Análise de Funil e Teste A/A/B

Projeto desenvolvido durante o bootcamp de Data Analytics da TripleTen.

## Objetivo

Analisar o comportamento dos utilizadores de uma aplicação alimentar através de um funil de vendas e avaliar os resultados de um teste A/A/B.

## Descrição

Neste projeto foram realizadas:
- análise do funil de conversão;
- estudo do comportamento dos utilizadores na aplicação;
- identificação de etapas com maior perda de utilizadores;
- preparação e limpeza dos dados;
- análise estatística de teste A/A/B;
- comparação entre grupos de controlo e teste;
- avaliação do impacto da alteração de fontes no aplicativo;
- interpretação de resultados e tomada de decisão baseada em dados.

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Jupyter Notebook
